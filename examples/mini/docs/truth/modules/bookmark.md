# 书签模块 · 技术设计

## 数据模型

单 JSON 文件（D-001）：`{ url: { title, tags[] } }`，url 即主键。

## 命令

| 命令 | 行为 |
|---|---|
| `bm add <url> [标题] [--tag a,b]` | 重复 URL 报错；标题缺省取域名 |
| `bm rm <id>` | 不存在报错；删除前确认 |
| `bm find --tag 工作` | 多标签取交集（i02 在做，T-003） |

## 约束

- 本地单机，无网络依赖；多端同步明确不做（D-002）。
