# 书签管家 PRD v1.1（仿真件）

> 现实中这份文档活在飞书 / Word 里；这里放一份仿真件让示例的"原文锚点"有处可指。

## §1 背景

命令行书签管理小工具，个人使用，本地存储。

## §2 基础功能

### §2.1 添加书签

`bm add <url> [标题]`：URL 必填，标题缺省取域名；重复 URL 报错。

### §2.2 删除书签

`bm rm <id>`：按 ID 删除；ID 不存在报错，删除前需确认。

## §3 标签（v1.1 新增）

添加时可带 `--tag 工作,学习` 多标签；`bm find --tag 工作` 按标签搜索，多标签取交集。

## §4 导入导出（草案，未定稿）

期望支持 Chrome 书签 HTML 的导入与导出。细节待产品补充。

## §5 多端同步（暂缓）

云同步诉求存在，但个人工具优先级低，暂缓。
